Bot written in python using the selenium package to send and reply to tweets. Tweets about news selecting different categories getting the news from a api. Also get the nasa picture of the day from the nasa api and can tweet that.
If the a user tweets 'spotify YYYY-MM-DD' replies with the spotify link to the number 1 on that date. If a user tweets 'birthday YYYY-MM-DD' sends birthday tweet on that date.
** minor bugs due to twitter making website changes. Also currently only runs from terminal, gets bot detected running off heroku server. **
